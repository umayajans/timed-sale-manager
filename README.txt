Easily set up time-limited sales for your WooCommerce products with automatic price changes.

== Description ==

Timed Sale Manager is a powerful WooCommerce extension that allows you to create time-limited sales for your products. Set a sale price along with start and end dates, and let the plugin handle the price changes automatically.

= Key Features =

* Set timed sales with specific start and end dates
* Automatic price changes at scheduled times
* Easy-to-use interface in the product editor
* Support for different time zones
* Fully translatable
* Compatible with the latest WordPress and WooCommerce versions

= Perfect For =

* Flash sales
* Holiday promotions
* Limited-time offers
* Seasonal discounts
* Special event sales

= Pro Features (Coming Soon) =

* Bulk sale scheduling
* Variable product support
* Email notifications
* Sale countdown timer
* Advanced reporting

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/timed-sale-manager` directory, or install the plugin through the WordPress plugins screen directly
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Make sure WooCommerce is installed and activated
4. Navigate to any product edit page to find the new "Timed Sale" tab

= Minimum Requirements =

* WordPress 5.2 or greater
* WooCommerce 6.0 or greater
* PHP version 7.2 or greater
* MySQL version 5.6 or greater

== Frequently Asked Questions ==

= Does this work with variable products? =

Currently, the plugin supports simple products only. Support for variable products is planned for a future release.

= Will the prices change automatically? =

Yes, prices will change automatically at the specified times without requiring any manual intervention.

= What happens when a sale ends? =

The product automatically returns to its regular price when the sale period ends.

= Does this work with my timezone? =

Yes, the plugin uses your WordPress timezone settings for all date and time calculations.

= Can I translate the plugin? =

Yes, the plugin is fully translatable and includes a .pot file in the languages folder.
